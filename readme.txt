== Description ==

Launch a blazing-fast, pixel-perfect website with the Noon WordPress block theme! Noon features over 50 beautiful pattern designs, 7 full-page pattern layouts, and a fully-customizable design system with Global Styles. Noon integrates seamlessly with all of the powerful new WordPress editor features, giving you the most lightweight and powerful website builder on the planet — no expensive page builder plugin required! ✶ Full demo: https://demo.noonwp.com ✶

== Installation ==

1. Go to Themes > Add New, search for "Noon" and then click "Install".
2. After installaion is done click on "Activate".
3. Enjoy!
